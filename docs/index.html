Demo of Live Speech and Emotion Recognition

The purpose of this demo is to showcase live speech and emotion recognition, converting them into text.

Demo Video

Watch the Demo Video

Live demo

https://speech-and-emotion-recognition.vercel.app/

Demo Features

  • Capture streaming audio using Deepgram Streaming Speech to Text.
  • Emotion recognition powered by face-api, built on top of the TensorFlow.js core API.

Quickstart

Clone the repository

Go to GitHub and clone the repository.

Install dependencies

npm install

Run the application

Once running, you can access the application in your browser.

npm run dev
